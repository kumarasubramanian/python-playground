# Employee Directory

## Overview

A beginner Python project that stores employee information using dictionaries.

The application allows users to add, search, update, delete, and display employee records through a simple command-line interface.

---

## Learning Objectives

This project focuses on learning Python collections, especially dictionaries.

---

## Features

- Add Employee
- Search Employee
- Update Salary
- Delete Employee
- Display All Employees

---

## Python Concepts Covered

- Variables
- Strings
- Integers
- Dictionaries
- User Input
- if / elif / else
- Dictionary Methods
- f-Strings
